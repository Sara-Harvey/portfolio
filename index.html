React Modal

Building a modal for users to create a React object was one of the bigger tasks of my 2021 internship with the educational startup, Mucktracker. I blogged about building alongside the team, and my tech takeaways. Check out my toy modal on CodeSandbox, too.

Mortality Report

Lets citizens easily browse a 128-page city report on death rate by NYC neighborhood. PDFs aren't formatted for data extraction, but Ruby's PDF::Reader::Turtletext gem and slice function made it possible. Visit the interactive report.
